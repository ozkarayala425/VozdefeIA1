# VozdefeIA1# ✝️ VozdefeIA1

VozdefeIA1 es una aplicación web de inteligencia artificial con enfoque cristiano, diseñada para brindar respuestas basadas en principios bíblicos, reflexión espiritual y acompañamiento en la fe.

## 📖 Propósito
La aplicación busca acercar la Palabra de Dios a las personas mediante tecnología moderna, permitiendo conversaciones interactivas con una IA que responde con mensajes inspirados en la Biblia, reflexión y esperanza.

## 🤖 Funcionalidades principales
- Chat interactivo estilo mensajería
- Respuestas generadas por inteligencia artificial
- Enfoque cristiano en cada respuesta
- Interfaz simple y tipo aplicación móvil
- Base preparada para futuras funciones de voz (texto a voz)

## 🌐 Tecnologías utilizadas
- HTML5 (interfaz)
- CSS3 (diseño)
- JavaScript (interacción)
- Node.js (backend)
- OpenAI API (inteligencia artificial)

## 🙏 Enfoque espiritual
VozdefeIA1 no reemplaza la Biblia ni la oración, sino que sirve como una herramienta de apoyo para reflexionar, aprender y fortalecer la fe en Dios.

> “La fe viene por el oír, y el oír por la palabra de Dios.” – Romanos 10:17

## 🚀 Estado del proyecto
En desarrollo activo. Nuevas funciones como voz, mejoras de diseño y expansión de respuestas bíblicas serán añadidas progresivamente.
